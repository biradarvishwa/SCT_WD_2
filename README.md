roject Title:
Interactive Stopwatch Web Application

Description:
The Interactive Stopwatch Web Application is a modern, user-friendly tool designed to accurately measure and record time intervals. Built with HTML, CSS, and JavaScript, it offers smooth animations, stylish UI, and advanced functionalities. Users can start, pause, reset, and record lap times with ease, making it ideal for sports timing, productivity tracking, or personal time management.

Key Features:
Digital Clock Interface with glowing effect when running.

Start, Pause, Reset, and Lap functionality for precise time control.

Lap Time List to track multiple intervals in a session.

Dark/Light Theme Toggle for personalized appearance.

Sound Alerts on key actions for better user feedback.

Persistent Lap Storage using browser localStorage to save lap times even after refreshing.

Responsive Design for desktop, tablet, and mobile compatibility.

Technologies Used:
HTML5 – Structure and layout.

CSS3 – Styling, animations, and responsive design.

JavaScript (ES6) – Stopwatch logic, event handling, and data persistence.

LocalStorage API – Storing lap times locally.

Google Fonts (Orbitron) – Digital display style.

Use Cases:
Sports & Fitness: Track laps, sets, or workout durations.
Productivity: Monitor work sessions and break intervals.

Education: Time quizzes, experiments, or presentations.
